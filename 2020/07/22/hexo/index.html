Posted on 2020-07-22 Edited on 2025-04-02

Intro

Hexo is a static blog framework built on Node.js. It has few dependencies and is very easy to use. Using it we can create a static website and deploy it to hosting platforms such as github, or in this case, we deploy it using Peanut Hull DDNS provider.
